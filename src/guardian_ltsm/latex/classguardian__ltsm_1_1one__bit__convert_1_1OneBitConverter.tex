\doxysubsection{Detailed Description}
\begin{DoxyVerb} `OneBitConverter` is the main GUI component 
for converting between 1-bit images and raw byte data.
It provides an interface for users to load images or 
paste data, adjust settings, preview results\end{DoxyVerb}
